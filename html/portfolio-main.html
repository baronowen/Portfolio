IPASS Web Application

Project summary

IPASS stands for Individual Propeduese Assesment.
In this assignment i made a web app in which a company could track the openings of a door,
when it's opened by an employee with a chip like key.
The core functionalities are adding a key, seeing who is in the building right now,
seeing when someone enterd and left the building, and adding an employee.

The opening of a door i simulated with a simple console application in which you typed your key code,
the application registers wether you entered or left the building,
and wrote the information with the time of entering or leaving to a database.

Source code for this assignment can be found on my GitHub at:
https://github.com/baronowen
